# A very trivial example how to require login

For some apps, role based security might be an overkill. Just storing something to
parent layout or to some session specific store can be enough.

To run the project, run `mvn jetty:run` and open [http://localhost:8080](http://localhost:8080) in browser.

To update to the latest available Vaadin release, issue `mvn 
versions:update-properties`
